# 🐍 Python Mini Projects Collection

A collection of beginner-friendly Python projects developed to practice core Python programming concepts. These projects focus on improving problem-solving skills through simple, interactive command-line applications.

## 📚 Projects Included

### 🧮 Calculator

A command-line calculator that performs basic arithmetic operations.

**Features**

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Exponentiation
* Input validation
* Handles division by zero

---

### ✅ To-Do List

A simple task management application that helps users organize their daily tasks.

**Features**

* Add new tasks
* View all tasks
* Edit tasks
* Mark tasks as completed
* Delete tasks
* Search tasks
* Save tasks using JSON
* Display completed and pending task statistics

---

### 🎮 Rock-Paper-Scissors Game

A fun command-line game where the user competes against the computer.

**Features**

* Random computer choice
* Score tracking
* Multiple rounds
* Winner announcement
* User-friendly interface
* Input validation

---

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/your-username/Python-Mini-Projects.git
```

### Navigate to the project folder

```bash
cd Python-Mini-Projects
```

### Run any project

**Calculator**

```bash
python Calculator.py
```

**To-Do List**

```bash
python ToDo_List.py
```

**Rock-Paper-Scissors Game**

```bash
python Rock_Paper_Scissors.py
```
